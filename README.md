### Додатково реалізовані механіки та покращення гри:

* **Критичний стан гравця.** Реалізовано візуальні ефекти (Particles) для танків, якщо їхнє здоров’я стає **<40%**. А також змінено висоту тону звука, що змінюється відповідно до здоров’я. Після знищення танку ці ефекти зникають.
* **Випадкові підсилювачі.** Впроваджено появу випадкових підсилювачів на мапі, які надають гравцям перевагу в бою та стимулюють постійний рух.
* **“Місячний лід”.** При наїзді на який танк сповільнюється (**speed * 0.7**). Після з’їзду з нього — швидкість повертається.
* **“Токсичні бомби”.** Бомби детонують при прямому зіткненні з танком або при влучанні снаряда (що знешкоджує “бомбу”), наносячи суттєвий урон (**-50 здоров’я**). Супроводжується відповідним звуковим та візуальним ефектом.
* **Портали-телепорти.** Реалізовано систему миттєвого просторового переміщення. Гравці можуть використовувати портали для швидкої зміни позиції, що додає грі стратегічної глибини.
